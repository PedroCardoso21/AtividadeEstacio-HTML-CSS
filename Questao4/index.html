Semântica

O que significa semântica?

Com relação ao HTML, a semântica tem como objetivo descrever o significado do conteúdo presente nos documentos, facilitando assim o entendimento de outra pessoa que vá o ler, ou até mesmo da pessoa que o fez.

Por que a semântica é tão importante em HTML5?

Principalmente para facilitar o entendimento de quem lê o documento HTML, existem diversas tags semânticas incorporadas nessa linguagem, existem dois tipos, semântica estrutural e a semântica no nível de texto

Semântica Estrutural:

Tag Função
header Representa o cabeçalho de um documento HTML
section Representa uma seção no documento
footer Representa o rodapé de um documento HTML

Semântica em nível de texto:

Tag Função
a Descreve um link no documento HTML
strong Destaca uma parte do texto no documento HTML
em Enfatiza uma parte do texto no documento HTML
