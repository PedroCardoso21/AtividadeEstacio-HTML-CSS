Tipos de linguagem de programação

Server Side

Também conhecido como back-end, estas são as linguagens que o servidor entende, ou seja, são processadas no próprio servidor. Responsável por processar dados, acessar banco de dados e gerar respostas enviadas ao navegador do cliente. Alguns exemplos são:

  • Python
  • PHP
  • Java

Client Side

Também conhecido como front-end, as linguagens client side são processadas pelo navegador, ela não tem acesso ao servidor. Responsável pela interação direta com o usuário e pela manipulação de interface do mesmo.
Alguns exemplos são:

  • JavaScript
  • HTML
  • CSS
