Um site que auxilia bastante no aprendizado de CSS, é o Flexboxfroggy. Este site consiste em ajudar um sapo a chegar no seu objetivo, nesse caso uma vitória-régia, utilizando diversas propriedades de CSS, como "justify-content", "align-items", entre outros. Auxiliando, através do estímulo visual, o aprendizado de uma forma divertida e intuitiva. Controlando o alinhamento, espaçamento, o usuário será capaz de compreender os conceitos de Flexbox muito mais facilmente.
